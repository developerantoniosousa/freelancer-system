# Freelancer

### Setup

1. Get a project.

```
$ git clone https://github.com/developerantoniosousa/freelancer-system
```

2. Install dependencies.

```
$ yarn install \\ npm i
```

3. Set up your environment

```
$ cp .env.example .env
```

4. Run migrations

```
$ npx sequelize db:migrate
```

5. Run

```
$ yarn dev \\ or npm run dev
```
